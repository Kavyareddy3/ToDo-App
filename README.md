# Java GUI – To-Do List App

# Objective
A simple To-Do List application built using **Java Swing**. Users can add tasks and delete selected tasks in a GUI window.
# Tools Used
- Java (JDK)
- Swing (JFrame, JButton, JTextField, JList)
- IntelliJ IDEA / Eclipse
# Features
- Add new tasks using a text field and button.
- Display tasks in a scrollable list.
- Delete selected tasks using a button.
- Simple and clean GUI layout.
# How It Works
1. Enter a task in the text field.
2. Click **"Add Task"** to insert it into the list.
3. Select any task and click **"Delete Task"** to remove it.
